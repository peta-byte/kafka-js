KafkaJS - Apache Kafka Client for Node.js

quick & basic with docker-compose

Leverage quickstart steps from KafkaJS docs and bitnami/kafka

Use the files as is to run a zookeeper instance, 1 broker, a publisher, and a consumer in docker containers

Hit the POST endpoint with a message query param to send new events

Go to localhost:4000/ from a browser to see these deets again :)
